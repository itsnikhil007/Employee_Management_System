# Employee Management System

## 📌 Project Overview

A Java Swing-based desktop application for managing employee records with login authentication and MySQL database integration.

---

## 🔐 Login Page

The system starts with a secure login screen where users must enter valid credentials to access the application.

![image](https://github.com/itsnikhil007/Employee_Management_System/blob/c5627568f5a6af00cdbbc50dc0ec39538ef2378a/EMPLOYEE_MANAGEMENT_SYSTEM/PROJECT_PICS/Login.png)

---

## 🏠 Dashboard (Main Screen)

After login, the dashboard provides options to manage employees:

* Add Employee
* View Employee
* Remove Employee

![image](https://github.com/itsnikhil007/Employee_Management_System/blob/c5627568f5a6af00cdbbc50dc0ec39538ef2378a/EMPLOYEE_MANAGEMENT_SYSTEM/PROJECT_PICS/Dashboard.png)
